使用Hexo和Gitee Pages搭建个人博客

发表于 2020-09-05 更新于 2020-09-07 分类于 博客

最近想捣鼓一下个人博客，以用来承载自己的一些想法，记录自己的成长，于是想到了用Hexo+GiteePages来搭建自己的个人博客。Hexo是一款快速、简洁且高效的博客框架，而选择GiteePages是因为Gitee是国内代码托管平台，访问速度比Github要快。

7、部署配置

8、启用Gitee Pages服务

点击仓库服务按钮，启动Gitee Pages服务
此时再访问 https://your-account.gitee.io 这个网址就能访问你的博客了。
